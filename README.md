# Mood Tracker

A simple application to track your daily mood.
This app was built with `npm create vite@latest . -- --template react-ts`.

## Features

- **Daily Mood Logging:** Quickly log your mood for the day.
- **Notes:** Add context to your mood entries with personal notes.
- **Mood History:** Visualize your mood patterns over time.

## Tech Stack

- **Vite:** A fast frontend build tool.
- **React:** A library for building user interfaces.
- **TypeScript:** A typed superset of JavaScript.
- **ESLint:** For code linting.
- **React Compiler:** An experimental compiler to optimize React apps.

## Getting Started

### Prerequisites

Make sure you have Node.js installed.

### Installation

1.  **Install dependencies:**
    npm install

2.  **Run the development server:**
    npm run dev

## Configuration

- **ESLint:** The project uses ESLint for code quality, configured in `eslint.config.js`.
- **React Compiler:** The project uses the experimental React Compiler for optimization, enabled in `vite.config.ts`.
